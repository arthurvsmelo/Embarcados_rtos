*****************************************************************************
** ChibiOS/RT port for ARM-Cortex-M3 STM32F103 on Bluepill board           **
*****************************************************************************

** TARGET **

The demo runs on the Bluepill board.

To build:

> make

** Flashing Target Board **

To flash the firmware to the bluepill, you can use STLink Utility.
Build the files using make command and flash the .bin file.
